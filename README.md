# TableXport

TableXport is the fastest way to extract and convert tables from any source into your favorite spreadsheet format.

## Features

- **One-Click Conversion:** Instantly grab table data from your clipboard.
- **AI-Ready:** Automatically cleans and sanitizes Markdown tables from ChatGPT, Claude, Gemini, and DeepSeek.
- **Multiple Formats:** Seamlessly export to Google Sheets, CSV, Excel, or branded PDF.
- **Cross-Platform:** Works with data from PDFs, websites, and desktop apps.

## How it works

1. Copy any table data to your clipboard (Ctrl+C).
2. Run the **Convert Table from Clipboard** command in Raycast.
3. TableXport will open in your browser with the data already parsed and ready for export.
